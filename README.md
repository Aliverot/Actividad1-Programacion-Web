# Actividad 1 - Ejercicios Prácticos de Programación Web

## Descripción
Este repositorio contiene los ejercicios prácticos desarrollados en clase, correspondientes al Tema 2 de la asignatura de Programación Web. El objetivo principal de esta actividad es integrar los ejercicios en un solo repositorio y publicarlos en línea mediante GitHub Pages como evidencia del trabajo realizado.

## Contenido del Repositorio
Se incluye un archivo `index.html` en la raíz del repositorio que funciona como un menú de navegación central. A través de este menú se puede acceder a los ejercicios:

1. **Ejercicio 1:** Estructura básica HTML5 ("Hola Mundo").
2. **Ejercicio 2:** Formato básico de texto y párrafos (`<p>`, `<strong>`, `<em>`).
3. **Ejercicio 3:** Implementación de enlaces simples (`<a>`).
4. **Ejercicio 4:** Inserción de imágenes (`<img>`).
5. **Ejercicio 5:** Integración combinada de texto, enlaces e imágenes.
6. **Ejercicio 6:** Uso de secciones semánticas (`<header>`, `<section>`, `<footer>`).
7. **Ejercicio 7:** Diseño de listas anidadas (Ejemplo de receta).
8. **Ejercicio 8:** Estructuración de datos con tablas (Horario de clases).
9. **Ejercicio 9:** Diseño de un formulario de contacto aplicando variables CSS, Flexbox y Grid.
10. **Ejercicio 10:** Calculadora de conversión de grados Celsius a Fahrenheit utilizando JavaScript básico.
11. **Ejercicio 11:** Convertidor de kilómetros a millas con validaciones numéricas y campos de solo lectura (`readonly`).
12. **Ejercicio 12:** Calculadora de conversión de pesos mexicanos (MXN) a dólares (USD) mediante una tasa de cambio predefinida, mostrando el resultado en un campo bloqueado (`readonly`).
13. **Ejercicio 13:** Validador de edad para determinar si una persona es mayor o igual a 18 años y puede votar, devolviendo el mensaje en una caja de texto de solo lectura (`readonly`).
14. **Ejercicio 14:** Analizador de números (mayor, menor, promedio) utilizando manipulación de arreglos en JavaScript (`split()`, `map()`, `reduce()`).
15. **Ejercicio 15:** Sistema de registro y análisis de estudiantes (promedio y calificaciones extremas) mediante el manejo de arreglos y objetos.

## Tecnologías y Conceptos Implementados
* **HTML5:** Estructuración semántica, enlaces, imágenes, listas, tablas, formularios y atributos de control de datos (`readonly`, validaciones `required`).
* **CSS3:** Incorporación de variables globales, Box Model, pseudoclases, Flexbox y Grid.
* **JavaScript:** Manipulación del DOM, validaciones lógicas (campos vacíos, `isNaN`), funciones matemáticas (`Math.max`, `Math.min`), operaciones con arreglos y gestión de objetos.
